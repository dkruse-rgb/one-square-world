# Tile Assets

Put the main game-ready tile PNGs here:

- `grass_tile.png`
- `forest_tile.png`
- `rock_tile.png`
- `water_tile.png`
- `dirt_tile.png`
- `farm_tile.png`
- `mountain_tile.png`
- `road_tile.png`

Recommended game paths:

```js
const TILE_IMAGES = {
  grass: "assets/tiles/grass_tile.png",
  forest: "assets/tiles/forest_tile.png",
  rock: "assets/tiles/rock_tile.png",
  water: "assets/tiles/water_tile.png",
  dirt: "assets/tiles/dirt_tile.png",
  farm: "assets/tiles/farm_tile.png",
  mountain: "assets/tiles/mountain_tile.png",
  road: "assets/tiles/road_tile.png"
};
```
